# План

Общий план выглядит следующем образом:

* Озвучить реплики по принципу «один аудиофайл — одна реплика»
* Нормализовать уровень громкости между участниками озвучки
* Вклинить озвученные реплики прям в игру
* Поставить фоном музыку из обзоров Аутскаута
* Запустить игру на автопроизведение
* Записать игровой процесс программой записи с экрана
* Выложить на канал Аутскаута в VK и на YouTube
* Распространить:
  * В Лапкосборе тизером
  * В [IFне Дискорда на канале #видео-по-играм](https://discord.com/channels/373163463842725891/1193125016469000222) прямыми ссылками на видео

## Озвучка

* Взять свой список пронумерованных реплик: [внутренний голос Павла](./scenario.md/#внутренний-голос-павла), [его внешний голос](./scenario.md/#павел), [голос девушки](./scenario.md/#незнакомка)
* Выбрать один из способов записи
* Озвучить реплику
* Назвать файл номером реплики
* Отправить записанное мне в Телегу

Можно по одной записанной реплике мне отправлять. Так даже лучше и сразу можно обсудить.

### Способы записи

Записать можно через смартфон или компухтер.

> **⚠️ Важно**
> Коней на переправе не меняем! Начали записывать с одного устройства — оставшиеся реплики записываем на него.
